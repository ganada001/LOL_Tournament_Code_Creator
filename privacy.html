개인정보 처리방침 (Privacy Policy)

LOL Tournament Code Creator("본 도구")는 사용자의 개인정보 및 Riot Games의 보안 정책을 최우선으로 생각합니다.

1. 수집하는 개인정보 항목

본 도구는 사용자의 API Key를 포함한 어떠한 개인 데이터도 수집, 요구 또는 로컬에 저장하지 않습니다.

보안 아키텍처: 모든 Riot API 요청은 개발자의 보안 백엔드(Google Apps Script)를 통해 처리되며, 사용자에게 API 키를 묻지 않습니다. 사용자의 데이터는 서버측에 일절 보관되지 않으며 실시간 요청 처리 후 즉시 소멸됩니다.

  • Discord Webhook URL: 사용자가 설정한 디스코드 웹훅 주소는 로컬에 저장되며, 알림 전송 목적으로만 사용됩니다.

2. 개인정보의 이용 목적

수집된 정보는 오직 다음의 목적으로만 사용됩니다.

  • 보안 백엔드를 통한 Riot API 토너먼트 코드 생성
  • 디스코드 채널로의 경기 정보 알림 전송

3. 데이터 보안

본 도구는 오픈 소스 소프트웨어이며, 모든 데이터 처리는 사용자의 로컬 컴퓨터에서 이루어집니다. API 키는 서버 측에서만 관리되며, 사용자에게 노출되지 않습니다.

4. 제3자 제공

본 도구는 수집된 정보를 제3자에게 판매하거나 공유하지 않습니다. Riot Games API 호출 시 Riot Games의 정책에 따라 데이터가 처리될 수 있습니다.

5. 문의처

본 정책에 대한 문의사항은 본 GitHub 저장소의 Issue를 통해 남겨주시기 바랍니다.
